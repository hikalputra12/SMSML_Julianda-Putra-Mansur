Deskripsi Singkat:
Implementasi lengkap pipeline machine learning dari preprocessing data,
model training dan tuning hyperparameter menggunakan MLflow dan remote tracking DagsHub,
Workflow CI/CD (GitHub Actions) untuk otomatisasi build Docker image ke Docker Hub,
serta monitoring real-time menggunakan Prometheus Exporter Proxy, Prometheus Server,
dan dashboard Grafana dengan minimal 10 matriks dan 3 alerting rules.
